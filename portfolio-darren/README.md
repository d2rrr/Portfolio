# Portfolio Darren DE OLIVEIRA -- DA SILVA

Portfolio web statique destiné à présenter le profil, le parcours, les compétences, les projets et les informations de contact de Darren DE OLIVEIRA -- DA SILVA.

## Technologies utilisées

- HTML
- CSS
- JavaScript simple

## Structure des fichiers

```text
portfolio-darren/
├── index.html
├── a-propos.html
├── projets.html
├── css/
│   └── style.css
├── js/
│   └── main.js
├── projets/
│   ├── kart-helice.html
│   ├── thermometre-bain.html
│   ├── prothese-main.html
│   └── pyroguard.html
├── assets/
│   ├── images/
│   ├── videos/
│   ├── icons/
│   └── documents/
└── README.md
```

## Pages

- `index.html` : accueil, présentation courte, compétences, projets, contact.
- `a-propos.html` : profil, parcours, formation, double diplôme, centres d’intérêt à compléter.
- `projets.html` : liste des projets BUT GEII et DU TSS.
- `projets/*.html` : pages détaillées de chaque projet.
